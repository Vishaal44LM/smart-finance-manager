

# SMART FINANCE MANAGER 💰

A personal finance management app for Indian users with all values in ₹ (Indian Rupees).

## Layout & Navigation
- Collapsible sidebar with icons and labels for: Dashboard, Expense Splitter, Trip Tracker, Budget Checker, Savings Goal, Subscriptions
- Clean header with app title and sidebar trigger
- Mobile-responsive layout

## Pages & Features

### 1. Dashboard
- Summary cards: Total Monthly Expenses, Remaining Budget, Total Subscription Cost, Savings Goal Progress
- Pie chart showing expense category distribution (using Recharts)
- Quick action buttons to each module

### 2. Personal Expense Splitter (💸)
- Input total bill amount in ₹
- Add/remove participant names dynamically
- Calculate and display equal split in a clean table
- Option to clear/reset

### 3. Trip Expense Tracker (✈️)
- Create trips with name and members
- Add expenses with description, amount, and "paid by" selector
- Expense history list per trip
- Auto-calculate balances (who owes whom)

### 4. Monthly Budget Checker (📊)
- Input monthly income
- Add expenses by category (Food, Transport, Shopping, Bills, Entertainment)
- Show total expenses, remaining balance
- Warning banner when overspent
- Pie chart of expense distribution

### 5. Savings Goal Calculator (🎯)
- Input target amount and deadline date
- Calculate and display daily savings needed
- Progress indicator

### 6. Subscription Cost Tracker (📺)
- Add subscriptions with name and monthly cost
- Preset suggestions (Netflix, Spotify, etc.)
- Show total monthly and yearly cost
- List with delete option

## Data Storage
- All data stored in localStorage for persistence across sessions

## Design
- Soft blue/green/white color palette
- Card-based UI with shadows and rounded corners
- Lucide icons throughout
- Fully responsive for mobile and desktop
